# Project Overview

GreenTask is a A hyperlocal climate-action micro-jobs marketplace connecting government officials (Gram Panchayat/local bodies) with community members in rural/climate-vulnerable regions to complete climate-resilience tasks for verified compensation.

This is GreenTask Mobile project. This is a Flutter app that is used by workers to apply to jobs and get paid! We need to implement relevant apis and screens to show all the required data and pass down proper data to backend. 

## Tech Stack
Framework: Flutter (latest stable version)
State Management: BLoC with Cubit architecture
Dependency Injection: GetIt
HTTP Client: Dio
Authentication: Supabase Auth (Google and Apple Sign-in)

## Plugins/Packages to be used:

- Dio for api calling
- flutter_svg for svg support
- bloc for state management and architecture
- get_it for dependency injection
- flutter_animate for microanimations
- google_fonts for fonts from google (montserrat and roboto)

any other plugins you think that is required should be installed from pub.dev website
